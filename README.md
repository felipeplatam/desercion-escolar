# desercion escolar:
Diseñé un proyecto de data storytelling analizando tasas
de deserción escolar de 33 departamentos colombianos (2015–2023) usando datos abiertos del MEN y el
DANE Desarrollé una aplicación web interactiva con Python, Streamlit y Altair con 5 secciones narrativas y
más de 10 visualizaciones interactivas Implementé técnicas de limpieza, transformación y visualización de
datos con Pandas y NumPy
